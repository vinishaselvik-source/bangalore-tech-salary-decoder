# Bangalore Tech Salary Decoder

### Data Analytics Project | Unlox Academy Live Project Session

## Overview

The **Bangalore Tech Salary Decoder** is a data analytics project developed as part of the **Unlox Academy Live Project Session**. This project focuses on analyzing salary trends among technology professionals in Bangalore by leveraging Python-based data analysis techniques.

The primary objective of this project is to derive meaningful insights from real-world employment data by examining the relationships between job roles, experience levels, technical skills, company types, and compensation structures.

---

## Project Objectives

* Analyze salary distributions across various technology roles.
* Examine the impact of professional experience on compensation.
* Evaluate salary variations across different company categories.
* Assess the influence of technical skills on earning potential.
* Generate business insights through exploratory data analysis.

## Dataset Description

The dataset contains information on **1,000+ technology professionals** working in Bangalore, including:

* Employee ID
* Job Role
* Years of Experience
* Current CTC
* Previous CTC
* Company Name
* Company Type
* Technical Skills
* Location
* Education Tier
* Joining Year
* Work Mode

---

## Methodology

The project was executed using the following analytical workflow:

### 1. Data Loading and Inspection

* Imported the dataset using Pandas.
* Examined data types, dimensions, and overall structure.

### 2. Data Cleaning and Preprocessing

* Standardized categorical values.
* Handled missing values.
* Converted salary values into a consistent numerical format.
* Removed inconsistencies and duplicates.

### 3. Exploratory Data Analysis (EDA)

The following analyses were performed:

#### Salary Analysis by Role

* Computed median compensation across various technology roles.

#### Experience-Based Salary Analysis

* Categorized professionals into experience bands.
* Analyzed compensation growth patterns.

#### Skills-Based Salary Analysis

* Compared salaries of professionals possessing AWS skills with those without AWS expertise.

#### Company Type Analysis

* Evaluated salary variations across:

  * Multinational Corporations (MNCs)
  * Unicorn Startups
  * Mid-sized Companies
  * Early-stage Startups

#### Statistical Analysis

* Generated descriptive statistics and summarized key trends.

---

## Key Findings

* Compensation generally increases with professional experience.
* Cloud computing skills, particularly AWS, positively influence salary levels.
* Company category significantly impacts compensation packages.
* Product and leadership-oriented roles demonstrate higher median salaries.
* Salary distributions vary considerably across organizational structures.
